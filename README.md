# Biolink Template

A sleek and customizable biolink page template for creators, developers, and anyone who wants to share multiple links in one place. Perfect for social media bios, portfolios, or personal landing pages.  

## Features

- ✅ Clean and modern design  
- ✅ Fully responsive layout (works on mobile & desktop)  
- ✅ Easy to customize colors, fonts, and links  
- ✅ Supports social icons and buttons
- ✅ Discord and Github intergration.
- ✅ Lightweight and fast-loading
